# Databricks Integration

This folder contains all files related to Databricks integration and setup.

## Files

- `databricks_config.py` - Configuration settings for Databricks connection
- `databricks_sql_commands.sql` - SQL commands for Databricks (full edition)
- `databricks_sql_commands_free_edition.sql` - SQL commands for Databricks (free edition)
- `databricks_sql_loader.py` - Python script to load data into Databricks
- `DATABRICKS_SQL_SETUP.txt` - Setup instructions for Databricks SQL

## Usage

Refer to the main project README for setup instructions and the DATABRICKS_SQL_SETUP.txt file for detailed configuration steps.
